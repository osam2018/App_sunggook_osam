#출타종합체계 개요
```
어플 이름은 출타종합체계입니다.  
현재 간부나 병사들은 출타신청을 할 때 양식에 맞추어서 손으로 일일이 쓰고 쓴것을 행정병이 옮겨적는 식입니다. 
스마트폰,앱웹이 활성화 되어있는 시대에서 이는 불편할수 밖에 없습니다. 
간부나 병사들이 각자의 스마트폰으로 어플을 설치하여 
출타를 캘린더형식에서 선택하여 서버로 전송하면 
서버에서는 개개인별로 DB에 저장해 간부,병사 개인의 정보를 가지고 있습니다.
또한 이 어플이 부대에서 실제로 사용되게 되면 부대 내 간부,병사들의 실제 출타,휴가 일수등의 정보를 알수있게 되므로
회원가입시 저절로 서버에서 간부,병사들의 정보를 클라이언트로 보내주게 됩니다.
따라서 실제 사용되는 어플이 될시에는 전군 간부 병사들의 휴가 관리를 할 수 있는 어플이 될 수 있고 관리자 웹페이를 생성해
출타 관리가 훨씬 용이해질 것입니다.
```

#본문
```우선 회원가입을 이름,소속,군번을 입력하고 아이디와 비밀번호를 만듭니다.
로그인을 한 후에는 출타신청,변경,삭제,국외여행신청,잔여휴가일수의 메뉴가 있습니다.
출타 신청과 변경은 달력을 클릭하는 방식으로 구현했으며 선택 한 후에는 직접 휴가의 종류와 날짜를 입력합니다.
이 정보들은 DB에 입력되어서 서버에서 개인별 휴가현황을 관리합니다. 따라서 휴가관리자들의 업무가 효율적이게 됩니다.
이 정보에 기반해 잔여휴가일수를 서버에서 보내줍니다. 추가적으로 국외여행신청은 입력폼을 통해 전송합니다.
휴가 신청을 한 정보는 출타양식으로 내 정보에서 확인할 수 있습니다. 이를 그대로 프린트할 수 있습니다.
```

#다른 휴가 관련 앱과 차별점
```
요번 대회에서 휴가관련 앱들이 많은 것으로 알고 있는데 이 앱들과 차이점은
서버를 이용하기 때문에 단지 개인의 스케쥴 관리,휴가증 관리만를 하는 어플이 아니라
전군 간부,병사들의 데이터를 서버에서 관리하고 이를 각 간부,병사들이 열람, 휴가 신청을 할 수 있게 해주는 어플리케이션입니다.
서버에서 전군의 휴가 데이터를 가지고 있으므로 각 병사들의 휴가관련 서류를 열람하고 양식대로 프린트 할수 있으므로 
서류 관리가 용이해지고 병사들이 휴가관련서류를 제출할필요가 없고 저절로 양식을 만들어서 등록되는 앱입니다.
```
